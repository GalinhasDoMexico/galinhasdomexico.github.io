Without JS

O projeto "Without JS" consiste em criar um website sem a utilização e JavaScript (ou qualquer outro tipo de script) seja via api ou próprio.

Este projeto não possui uma base ou objetivo, foi apenas criado para entretenimento e treino. Ainda pode ocorrer atualizações, mas este projeto não está como foco.
