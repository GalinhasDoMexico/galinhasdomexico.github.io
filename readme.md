## Olá!
Me chamo **GalinhasDoMexico**.

Atualmente estou aprendendo inglês e programação (principalmente para websites, neste momento) e estou tentando melhorar minhas renders de Minecraft.


Tudo começou com uma vontade de criar um website pessoal e aprender mais sobre o assunto, iniciei com um projeto simples de site, com uma página inicial e sobre mim.

Ao longo do tempo fui adicionando mais coisas, como um tema LGBT como easter egg, tema claro e escuro e até versão em inglês (en-us).


Depois pensei em refazer o site todo, seja os scripts, layout e tudo mais. O remake começou de uma forma mais simples na versão 0.1.0-remake. E junto a ela veio meu primeiro domínio "cofm.site" (cofm vem de ChickenOfMexico, meu username em inglês).

Mas percebi que o site ainda não estava da forma que queria e então refiz algumas coisas e assim nasceu a versão 0.2.0-remake.

Eu ainda tenho uma versão antiga do site, quem sabe eu lance separadamente.

### Instalação
> Lembre-se, este website está sob a **[Licença GNU GPL v3.0](LICENSE "Veja a Licença GNU GPL v3.0")**.

1. Clone o respositório:
   ```bash
   git clone https://github.com/GalinhasDoMexico/galinhasdomexico.github.io.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd galinhasdomexico.github.io
   ```
3. Abra o arquivo `index.html` em seu navegador.
